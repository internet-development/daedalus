# Beans Brainstorming Skill

## Purpose

Guide agents through creative exploration and requirements gathering for new features, epics, and milestones.

## Workflow

1. **Context Gathering**
   - Ask clarifying questions (one at a time, prefer multiple choice)
   - Understand the problem space and user needs

2. **Exploration**
   - Present ideas in 200-300 word sections
   - Validate understanding before proceeding

3. **Output**
   - Create beans for refined concepts
   - Link to parent epics/milestones as appropriate
